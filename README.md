# react-redux-boilerplate
A react & redux & react-router & webpack2 starter kit for react SPA project.

一个基于 reacr, redux, react-router, webpack2 的单页面应用脚手架。

## Usage

- npm install
- npm start 
- npm build


## 功能

- css module
- less, post-css
- react hot reload
- dll
- redux & react-router & redux-action & redux-promise
- webpack2 
- more
